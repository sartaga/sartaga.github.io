The process of testing each feature as an independent unit as described in “Design by Feature/Build By Feature” phase. If each feature is tested as much as possible at the time of implementation, the number of bugs should be significantly reduced by the System Test phase of the project. It is expected that it is not possible to completely test a feature at the time of implementation since dependent features might not yet be completed, but as much testing as possible should be done.<br/>Testing is managed by Visual Studio Testing Manager. Test Cases and expected outcomes should be integrated into the Feature Spec.<br/>
